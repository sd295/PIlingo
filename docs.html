PowerPoint & Excel Rendering

PLI 7 can generate structured data for presentations and spreadsheets. These require specific "Large Scale" UI components.

1. PowerPoint (<pptx>)

When this tag is present, the frontend should render a Full-Width Slide Deck.

<pptx>
[Slide 1]
Title: Q4 Financial Report
Content: - Revenue up 15% \n - New market entry
[Slide 2]
Title: User Growth
Content: - 2M new active users
</pptx>

Note: Presentations should be displayed in a prominent, large container with slide navigation controls.

2. Excel (<excel>)

Used for tabular data analysis. Render this inside an Interactive Data Grid at the end of the response.

<excel>
Month, Revenue, Growth
Jan, 12000, 2%
Feb, 15000, 5%
</excel>

Standard Formatting

PLI 7 uses standard Markdown for text. STRICT CONSTRAINT: The engine never uses acute accents (é, á, ó).
